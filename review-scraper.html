⚠ Browser CORS Limitation
SerpApi blocks direct browser calls. You need to run the scraper via the Node.js script below instead — it takes 2 minutes to set up and outputs a CSV you paste back here.
How it works: Run the Node.js script on your computer → it saves a reviews.json file → paste the contents into the JSON Import box below → AI analysis runs as normal.
📦 Node.js Script — Run This on Your Computer
1. Make sure Node.js is installed (nodejs.org)
2. Click Generate Script → copy it → save as scrape.mjs
3. Open terminal in same folder → run: node scrape.mjs
4. Copy the output JSON → paste into JSON Import box below
